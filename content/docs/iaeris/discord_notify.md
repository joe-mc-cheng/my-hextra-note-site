# 🎮 iAeris 系統：Discord Webhook 通知設置指南

透過 Discord 的 Webhook 功能，您可以發送 iAeris 系統的即時警報（如因子超標、設備離線），直接推送到指定的 Discord 頻道，實現跨平台的即時監控。

## 步驟 1：帳號準備與伺服器建立

在開始設定之前，您需要先準備好接收訊息的「空間」。

1.  **下載並註冊**：下載 Discord App 並註冊帳號。
2.  **電腦端登入**：建議使用電腦瀏覽器登入Discord官網，並以登入與APP相同的 Discord帳號(上列1.)。
3.  **建立伺服器**：
      * 點擊左側列表底部的「**+**」號（新增伺服器）。
      * 選擇「親自建立」並設定伺服器名稱（例如：`空氣品質通知中心`）。
      * 這是您專屬的私密空間，只有受邀者可以看見通知。

-----

## 步驟 2：建立並取得 Webhook 網址

Webhook 就像是一個「數位傳聲筒」，能讓 iAeris 系統把話傳進 Discord。

1.  **選擇頻道**：在您剛建立的伺服器中，找到預設的文字頻道 `#一般` (general)。
2.  **編輯頻道**：點擊頻道名稱旁的 **齒輪圖示 (編輯頻道)**。
3.  **整合設定**：
      * 從左側選單中選擇「**整合 (Integrations)**」。
      * 點擊「**建立 Webhook (Create Webhook)**」按鈕。
4.  **配置 Webhook**：
      * 點擊「**新 Webhook (New Webhook)**」進行設定。
      * **修改名稱**：您可以將其命名為 `iAeris通知` 或 `空品警報`，這會是訊息發送時顯示的人名。
      * **複製網址**：點擊下方醒目的「**複製 Webhook 網址**」按鈕。
5.  **⚠️ 重要安全性提醒**：
      * 請妥善保存這個 URL。**持有此網址的任何人都能發送訊息到該頻道**，請勿將其公開。

-----

## 步驟 3：在空品設備系統中設定

取得 Webhook 網址後，需將其填入 iAeris 管理系統完成對接。

1.  **進入管理頁面**：以管理權限登入 iAeris 本地雲端系統。
2.  **進入通知設定**：找到左側功能列 **空品上下限設定** > 選擇要發通知的設備 > 左側`設備通知上下限屬性` 
   * 先設定每個因子發通知的上下限規格(超出規格會發通知)。
3.  **填入 Webhook**：
      * 在 **Discord Webhook** 欄位中，貼上剛才複製的長串網址。
4.  **啟用通知**：
      * 將 **Discord 通知啟用** 欄位設定為 `True` (啟用)。
      * 確保勾選「更新設定」相關核取方塊。
5.  **確認儲存**：按下 **Submit** 按鈕。

## 🚀 測試與完成

  * **測試發送**：設定完成後，您可以透過系統的手動測試功能或等待下一次因子觸發警報，確認 Discord 頻道是否出現通知。
